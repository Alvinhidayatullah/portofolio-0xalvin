> **Personal portfolio website showcasing Web3 development, cybersecurity expertise, and professional certifications.**

## ✨ Features

- 🎨 **Modern Cyberpunk UI** - Eye-catching design with animated gradients and glassmorphism effects
- 🖥️ **Interactive Terminal** - Live command simulation showing developer info
- 📜 **Certificate Carousel** - Horizontal scrolling gallery of professional certifications
- 🧭 **Smooth Navigation** - Active section tracking with animated underline indicators
- 📱 **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- ⚡ **Performance Optimized** - Lazy loading, rate limiting, and efficient event handling
- 🔒 **Security Hardened** - Input sanitization and CSP headers implementation
